JavaScript Nedir

JavaScript web programlama dilidir. Daha çok dinamik ağırlıklı web siteleri yaparken kullanılmaktadır. JavaScript'in DOM(Document Object Model) ile HTML ve CSS müdahalesi bulunmaktadır. (DOM yapısı anlatılacaktır). En basit örnek olarak form doğrulama (validation), stil düzenleme, ekleme, silmeyi sağlıyor.

JavaScript günümüzde hızlı ilerleyen teknoloji ile back-end (arka uç), UI oluşturma (tanımlama), masaüstü programlama, yapay zeka, mobil programlama alanlarında da kullanılmaktadır. Bu teknoloji alanlarında JavaScript framework, library sayesinde projeler oluşturulmaktadır.

Tarihçe

JavaScript 1995 yılında Brendan Eich tarafından geliştirildi. JavaScript ilk zamanda adı sırasıyla Mocha, LiveScript Netscape ve Sun Microsystems tarafından JavaScript olarak isimlendirildi. Statik web sayfalarını dinamik ve etkileşim vermek için tasarlandı.

Netscape JavaScript projesini (Brendan Eich Netscape çalışanı) Ecma International tarafından oluşturulan standartlara göre geliştirmiştir. 2025 yılında yazılan bu yazı referans alanlarında ECMAScript çercevesinde Ecma International tarafından geliştirilmektedir.

Bilgi

Günümüzde web siteleri istemci ile etkileşim içindedir. Web teknolojisi gelişmeden önce istemci ile web sitesi etkileşimi fazla değildi. teknolojinin gelişmesi ile bu durum azalmıştır.
