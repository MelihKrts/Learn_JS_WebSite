JavaScript Kurulumu

JavaScript'in kendisi için kurulum gerekmez. JavaScript tarayıcılar tarafından çalıştırılan programlama dili olduğundan sistem (komut istemi) üzerinden çalıştırmak için Node.JS kurulumu gerekmektedir. 2025 Nisan tarihi itibari ile mevcut Node.Js LTS (Long Term Support) v22.14.0 sürümüdür.

Node.js buradan indirebilirsiniz. Node.js

Node.js kurulumu Windows için basittir. Dosya indirildikten sonra dosyayı çalıştırmanız ve yükleyici de açıklanan adımları takip etmeniz yeterlidir.

Kurulum Sonrası

Node.js kurulumunu başarılı şekilde tamamlandıysa Komut istemi (CMD) veya klavye kısayolu win + r cmd açılan ekrana node -v yazın. Versiyon numarası yazıyorsa kurulum tamamlanmıştır. 

v22.13.0

Node Js Test Etme

Node.Js kurulumu tamamlandıktan sonra Node.js çalıtırmak ve test etmek için masaüstüne veya istediğiniz herhangi bir dizine test klasörü oluşturup klasörün içine app.js dosyası oluşturup bu dosyaya şu kodu yazılmalı. 

console.log("Node.js başarıyla yüklendi. Bu kod Node.js aracılığıyla çalıştırıldı")

Dosya bulunduğu konum doğru belirtildiyse konsolda çıktı verecektir ve çıtkı şu şekilde olacaktır. 

Node.js başarıyla yüklendi. Bu kod Node.js aracılığıyla çalıştırıldı

Daha doğru ve gelişmiş kod yazmak için kod editörü kullanılması daha iyi olabilir. JavaScript (Web programlama) için bazı editörler kullanılabilir istediğinizi kullanabilirsiniz. Bunlar;

NVM Kurulumu Daha mı İyi ?

NVM (Node Version Manager) birden fazla Node.Js sürümü kurmayı ve yönetmektedir. Node.Js kurulumu Normal kurulum ziyade biraz uğraştırıcı ama sürüm yükseltme, düşürme, konusunda kolaylık sağlamaktadır. NVM ile Node.js kurmak için bu adımlar yapılmalıdır;

1. Gerekli Dosya İndirilmesi

Bu link üzerinden nvm-windows gerekli dosyayı indirmeniz ve çalıştırmanız gereklidir.

2. NVM Kontrolü ve Node.Js Kurulumu

İndirilen dosyayı çalıştırdıktan sonra NVM kurulumunu doğrulamak için komut istemine Aşağıda verilen kod yazılıp kontrol edilmeli. Versiyon numarası yazıyorsa NVM bilgisayarınıza kurulmuş olup NVM üzerinden Node.Js kurulumu yapabilirsiniz. NVM komutlarına buradan ulaşabilirsiniz.

nvm -v

Paket Yönetim Sistemi

NPM, Yarn ve PNPM paket yönetim sistemlerinin karşılaştırılması.

Paket Yönetim Sistemi, yazılım paketlerinin ve kütüphanelerinin konfigürasyonunu sağlar. JavaScript için paket yönetim sistemi başlıca NPM (Node.js kurulumu esnasında varsayılan olarak NPM kurulur), yarn ve PNPM'dir.

NPM (Node Packaage Manager)

NPM (Node Package Module) kısaltılmasıdır. Javascript için geliştirilmiş ve Node.js’in standart kabul ettiği paket yönetim sistemidir. NPM 2009 yılında oluşturulmuş, 12 Ocak 2010 yılında piyasaya sürülmüştür. Isaac Z. Schlueter tarafından yazılmıştır. NPM, Github tarafından satın alınmıştır.

Yarn

Yarn, Meta Inc(Facebook), Exponent, Google ve Tilde işbirliği sonucu Sebastian McKenzie tarafından Javascript için geliştirilmiş ve NPM paket yöneticisinin alternatifidir. Yarn önemli kılan özelliği NPM, kıyasla daha hızlı performans, bellek tasarrufu ve güvenlik sağlamasıdır.

PNPM

Performant NPM olarak adlandırılan PNPM, yarn ve NPM tüm özelliklerine sahip ve performanslı olan paket yönetim sistemidir. PNPM, versiyon 1 2017 yılında Zoltan Kochan tarafından yayınlandı. PNPM’yi Yarn ve NPM’den ayıran özelliği bir paket yönetildikten sonra önbelleğe alır. Sonraki projelerde kullanılacaksa yerelden çekilir böylece NPM üzerinden dosyayı tekrar indirmeye ve bellek harcamasına gerek kalmayacaktır.
