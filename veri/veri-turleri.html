Veri Türleri

Programlama dillerinde veri türleri vardır. Bu veri türleri değerin hangi türde tutacağını belirtmek için kullanılır. Bu veri türleri İlkel ve ilkel olmayan veri türü olmak üzere ikiye ayrılır.

İlkel (Primitive) Veri Türleri

İlkel Veri tipi değiştirilemeyen temel değerdir. Oluşturulduktan sonra değiştirilemez. Doğrudan bellekte (RAM) saklanmaktadır. İlkel (Primitive) veri türleri;

  • String
  • Number
  • BigInt
  • Boolean
  • Undefined
  • Null
  • Symbol
String

String veri türü metni temsil eder Örnek kullanım şu şekildedir.

let name = "John"
Number

Number veri türü sayısal veri türüdür. Örnek kullanım bu şekildedir.

let number = 10
Boolean

Boolean mantıksal veri türüdür. 2 değer alır. true ve false Durum olumlu ise true durum olumsuz ise false değeri döndürür 

let trueValue = 1 > 0
console.log(trueValue)
                            
// 1 sayısı sıfır sayısındaan büyüktür sonuç olumlu olduğu için true değeri döndürür.
                            
let falseValue = 1 > 2
console.log(falseValue)
                            
    
// 1 sayısı 2 sayısından küçüktür sonuç olumsuz false değeri döndürecektir.
Undefined

Undefined (tanımsız) bir değişken bildirilmiş ama değer almamıştır. 

let value
console.log(value)
Null

Hiçbir değeri yoktur. Boş veya var olmayan değer anlamı taşır. 

let value = null
console.log(value)

Non Primitive Data Types (İlkel Olmayan Veri Türleri)

İlkel olmayan veri türleri değiştirilebilirdir. Non-Primitive veri türleri;

  • Array
  • Object
Array

Array (Dizi) birden fazla değeri tek bir değişken altında toplamayı sağlar.

let arrayValue = ["Justice", "Law", "Equality"]
Object

Object (Nesne) birden fazla değeri depolayan yapıdır.

let objectValue = {
    name: "John",
    surname: "Doe",
    age: 36,
    hobbies: ["Read Book", "Swimming", "Play PC Game", "Walking"],
    fullName() {
        return `${this.name} ${this.surname}`
    }
}
                    
console.log(objectValue.fullName())
// Basit object kullanımı farklı veri türü kullanımına izin verir.

Bilgi Kutusu

Bilgi

İlkel olmayan veri ve İlkel veri türü değişmez özelliği gösterimi

let name = "John"
let newName = name

newName = "Brendan"

console.log(name)
console.log(newName)
                        
// name değeri John atandı.
// newName = name bu kod ile John değeri kopyalandı
// newName ile yeni değer atandı ama sadece newName etkilendi name değişkeni ilk değeri John'a sahip değer kopyalama yapıldı ilk değer değişmez
                            


let firstName = { name: "John" }
let newFirstName = firstName

firstName.name = "Brendan"

console.log(firstName.name)
console.log(newFirstName.name)


// firstName değişkeni nesne (object)
// newFirstName = firstName bu kod firstName değişkeni ile aynı değeri alıyor.
// iki değişkenin değeri aynı değere sahiptir.
// firstName değeri değişince, newFirstName değeri de değişmektedir.
